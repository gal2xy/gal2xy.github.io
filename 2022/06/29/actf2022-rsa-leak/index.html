b = randrange(0, pow(2,256))

qq = next_prime(q+rq)
if pp % pow(2, 4) == (pp-p) % pow(2, 4) and qq % pow(2, 4) == (qq-q) % pow(2, 4):
n = pp*qq
rp = pp-p
rq = qq-q
return n, rp, rq

法一

论文：8.pdf (upm.edu.my)

通过上述论文可知，我们要先求出rp(pp-p),rq(qq-q)。

令

\[ leak\_out = 90846368443479079691227824315092288065\\ leak\_n = 122146249659110799196678177080657779971\\ tmp = 0xdeadbeef \]

该题中leak_n很小易分解出leak_p,leak_q，且rp,rq的范围在[0,1<<24]左右，故可以对rp进行枚举（枚举范围可以减半，即rp,rq的值对调），有等式​，然后解密得到rq，通过以下条件筛选出rp,rq

\[ 1)rq\in[0,1<<24)\\ 2)rp*rq\equiv n\quad(mod\quad 16)\\ \]

然后再根据论文中的步骤写出算法：

法二

rp,rq的求法同上。（rp = pp - p，rq = qq-q）

\[ \begin{cases} pp = a^4+rp \\qq = b^4+rq \end{cases} \]

比较各数值大小

a,b在256bit左右，a^4,b^4在1024bit左右
而rp,rq在24bit左右
故n=pp*qq=(a^4+rp)*(b^4+rq)≈a^4*b^4

于是可以联立方程

\[ \begin{cases} n = (a^4+rp)*(b^4+rq) \\n^{1/4} = a*b \end{cases} \]

解此方程可得到a,b，继而求出p,q。
